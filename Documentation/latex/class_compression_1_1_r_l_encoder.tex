\doxysubsection{Detailed Description}
A class that provides methods for performing run-\/length encoding. 

As run-\/length encoding is the first step of the compression stack, the methods accept and return uint16\+\_\+t$\ast$. For converting to char$\ast$ the byte\+Encoder class should be used.
